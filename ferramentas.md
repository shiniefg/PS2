# HTML : Quais elementos estão sendo usados?

background-color interno para a coloração do site

h1, h2 e h3: para explicações no site

div : para manipulação dos elementos

img : adicionar uma imagem na pagina do site

br : para distanciar os elementos

p : para um paragrafo

a : para adicionar um link 

link rel="icon" : adicionar uma imagem ao site 

link rel="stylesheet" : conectar um css externo



# CSS : Quais propriedades estão sendo usados?

  *(seletor universal)
   "margin: 0;
    padding: 0;
    box-sizing: border-box" : para tirar as bordas do site

    text-align: center : para alinhar um elemento no centro, usado mais em h1 e h4

    background-color : coloração de elemento

    box-sizing: border-box : altera a forma e largura do elemento

    color : para coloração das palavras

    display: flex : transforma o container "flexivel"

    gap : distanciamento das imagens na pagina pxpersonagem.html e outros elementos tambem

    font-family : Mudar a fonte de escrita do elemento

    border : borda do elemento, usei para decidir a coloração dela

    padding: 26px 1px : modificar as bordas do elemento

    cursor: pointer : Muda a aparencia do curso ao passar pelo elemento que foi usado essa propriedade

    margin-top: distancia entre o elementoe o topo da pagina

    background-position : o posicionamento da imagem

    background-repeat: no-repeat : para não repetir a imagem

    button:hover : mudar a cor do butão ao passar o mause
    
    border: none : sumir com a borda do button

    border-radius : arrendondar as bordas do elemento

    background-image: url : selecionar a imagem

    {width: 100%;
    width: 1100px;
    height: 400px;} : proporções das imagens

    margin-left: 50px : posicionamento da imagem

    position: absolute : deixar "livre" o elemento ou em pocisão absoluta

    as "margin-" foram usadas para o distanciamento e posicionamento dos elementos do site
    
    background-size: cover : para dimensionar uma imagem de fundo de forma a cobrir todo o elemento contêiner

    z-index : propriedade para controla a ordem de empilhamento vertical de elementos sobrepostos
